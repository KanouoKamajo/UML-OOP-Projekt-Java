# 🎉 UML OOP – Partyplaner-Projekt (C#)

Dieses Projekt veranschaulicht die **objektorientierte Modellierung (OOP)** mit Hilfe von **UML-Diagrammen**.  
Ziel ist es, ein System zur Organisation von Partys oder Events zu entwerfen – mit sauberer Struktur, Klassen und Beziehungen.

---

## 📌 Inhalt des Projekts

- 🎯 **Anwendungsfall**: Planung & Organisation von Partys
- 👩‍💻 **Programmiersprache**: C#
- 🧠 **OOP-Konzepte**: Vererbung, Polymorphie, Kapselung
- 🧭 **Diagramme**: Klassendiagramm, Objektdiagramm, Sequenzdiagramm

---

- Umsetzung der objektorientierten
